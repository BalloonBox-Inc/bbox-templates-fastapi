# Application Tests

This directory contains all application tests, including Unit and API tests.
Learn more about Pytest at https://docs.pytest.org/en/7.1.x/how-to/index.html#how-to

## API Testing

The purpose of API Testing is to check the functionality, reliability, performance, and security of the programming interfaces.
Learn more about API testing at https://fastapi.tiangolo.com/tutorial/testing/

## Unit Testing

The purpose of Unit Testing is to test the smallest parts of the application where they are individually and independently scrutinized for proper operation.
Learn more about Unit testing at https://docs.python.org/3/library/unittest.html
